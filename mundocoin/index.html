FAQ

FREQUENTLY ASKED QUESTIONS

What is MundoCoin/MNO?

MundoCoin/MNO is a new L1 Coin that integrates WEB3.0 attributes such as transactions, social networking, and games.

What is MundoCoinNode/MCN?

MundoCoinNode/MCN is a credential, and only with MCN can you run the master node..

What is MundoCoin’s supply?

Total supply is 21,000,000. Current supply can be found on MNO blockchain explorer.

Is MundoCoin listed at any Exchange?

MNO is not yet listed on exchanges, but it will be soon.

What Wallet can I use to hold MundoCoin?

Can use PC wallet, mobile wallet to store and send MNO.

How to get MundoCoin?

You can choose to run the master node of the MNO through the server, or run the mini node through the mobile phone to obtain the MNO.
