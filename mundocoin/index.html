What is MundoCoin?

MundoCoin is an upgrade project of blockchain cryptocurrency technology, based on the network built by nodes, fully integrated into web3.0, to achieve intelligent interaction between network and people, people can refract virtual assets of web3.0 to reality.

Symbol: MNO
Total: 21 million
Participation mode: running node (server + fixed IP), MINI node (mobile phone)
Proof method: POC (Proof Of Cooperation)
Block reward: 10 MNO
Distribution mode: 4% enters the project fund pool, 1% enters the development pool, 25% is distributed equally by the master node, and 70% is distributed equally by the mini-node
Transaction cost: 0.001%
Block time: 120 seconds
Transaction speed: can reach 10000 transactions per second
